# 🌿 Nature Walks Tracker
## 📖 Overview
Nature Walks Tracker is an Android application designed to encourage mindfulness and outdoor exploration. The app presents users with a curated list of scenic walks while enhancing the experience with relaxing ambient nature sounds played through a foreground service.
Users can interact with the app by selecting walks they plan to complete each week, promoting balance, reflection, and connection with nature.

## ✨ Features
### 📍 Scenic Walk Listings
Browse a list of peaceful and visually appealing nature walks.

### ✅ Weekly Selection System
Users can select up to 2 walks per week
Selection is done using a long-press gesture
Visual feedback highlights selected walks

### 🎧 Ambient Sound Experience
Plays relaxing nature sounds (e.g., rain, forest, birds)
Runs using a foreground service
Enhances focus and immersion

### 🔔 Persistent Notification
Keeps the foreground service active
Allows users to control or monitor playback

### 🛠️ Technologies Used
Java / Kotlin (Android Development)
Android SDK
RecyclerView – for displaying the list of walks
Foreground Service – for continuous audio playback
MediaPlayer – for handling sound
Gesture Detection (Long Press) – for user interaction

### 📱 How It Works
Launch the app to view a list of available nature walks.
Long-press on a walk to select it.
You can select a maximum of 2 walks per week.
Selected items will be visually highlighted.
Ambient sounds automatically play in the background using a foreground service.
A persistent notification ensures the audio continues even if the app is minimized.
